# Frontend App

Next.js frontend application for RayZ.

## Getting Started

```bash
# Development
pnpm dev

# Build
pnpm build

# Start production server
pnpm start
```

## Features

- Next.js 14+ with App Router
- TypeScript
- TailwindCSS
- Shared UI components from `@rayz/ui`
- Shared types from `@rayz/types`
